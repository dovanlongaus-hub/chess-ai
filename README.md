# Chess AI Project

## Overview
This Chess AI project is designed to analyze and play chess games using artificial intelligence techniques. The aim is to develop a system that competes at various levels of play, providing a challenging experience for users.

## Features
- **Game Analysis**: Ability to analyze chess positions and give recommendations on optimal moves.
- **AI Opponent**: Play against the AI with multiple difficulty levels.
- **Opening Book**: Integrated opening book to enhance the AI's performance in the opening phase of the game.
- **User Interface**: Interactive UI to display the chessboard and allow users to make moves.
- **Multi-platform Support**: Compatible with web and desktop platforms.

## Technology Stack
- **Programming Language**: Python
- **Libraries**: TensorFlow, NumPy, Chess, Flask (for the web interface)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dovanlongaus-hub/chess-ai.git
   ```
2. Navigate to the project folder:
   ```bash
   cd chess-ai
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To start the application, run:
```bash
python app.py
```
